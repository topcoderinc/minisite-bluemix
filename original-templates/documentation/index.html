Delta - Responsive HTML5 Template

Overview

Delta is an HTML5 landing page template designed to help app developers promote their mobile apps. It is a good starting point to quickly have a web presence for your product to drive interest and sales. If you have any comments or suggestions please get in touch.

  • Author: 3rd Wave Media
  • Email: hello@3rdwavemedia.com
  • Twitter: 3rdwave_media

Page Structures

The template page consists of HTML Head, Page Header, Promo Section, Features Section, How it works section, and Page footer

Theme Colour Schemes

There are 10 colour schemes available for the template. You can easily change the colour scheme by changing the stylesheet.

Default

<!-- Theme CSS -->  
<link id="theme-style" rel="stylesheet" href="assets/css/styles.css">

If you want to change to a different colour scheme (eg. colour2), you just need to replace the current stylesheet file styles.css to styles-2.css

<!-- Theme CSS -->  
<link id="theme-style" rel="stylesheet" href="assets/css/styles-2.css">

Theme Style Switcher

The theme style switcher is only used for demonstration purpose. You probably want to remove the relevant code (HTML and Javascript) in your production site.

HTML

The HTML code for the style switcher is below the page footer in index.html

<!-- *****CONFIGURE STYLE****** -->  
<div class="config-wrapper">
    <div class="config-wrapper-inner">
        <a id="config-trigger" class="config-trigger" href="#"><i class="fa fa-cog"></i></a>
        <div id="config-panel" class="config-panel">
            ...
            ...
        </div><!--//configure-panel-->
    </div><!--//config-wrapper-inner-->
</div><!--//config-wrapper-->

Javascript

The Javascript code for the style switcher is in the bottom of assets/js/main.js

    /* ======= Style Switcher ======= */
    $('#config-trigger').on('click', function(e) {
        var $panel = $('#config-panel');
        var panelVisible = $('#config-panel').is(':visible');
        if (panelVisible) {
            $panel.hide();
        } else {
            $panel.show();
        }
        e.preventDefault();
    });
